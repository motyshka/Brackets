что такое "Brackets"?

Braсkets-это Свободный текстовый редактор для веб-разработчиков. Ориентирован на работу с HTML, CSS и JavaScript. Эти же технологии лежат в основе самого редактора, что обеспечивает его кроссплатформенность, то есть совместимость с операционными системами Mac, Windows и Linux.
